Personnel

The personnel involved with ReverseRecipe are the following:
  1. End Users
    1. Registered Users
    2. System Administrators
    3. Offline/Unregistered Users
  2. Software Engineers / System Developers
  3. Interested Outside Parties
End Users are those who use the ReverseRecipe for its intended purpose. Registered Users are those registered in the user database in one of the Reverse Recipe central host computers. Offline/Unregistered Users are those not in a the central host computer. System Administrators are those who perform overall system administration, as well as database managment. System Developers are those in charge of implementing the application and the release of the applications. The project is open source available on github to other Interested Outside Parties.
